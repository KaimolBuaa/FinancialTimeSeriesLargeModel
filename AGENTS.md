# Project Delivery Preference

When producing a Markdown document for this user, keep the Markdown file in
the local workspace and also create or update a corresponding Feishu cloud
document. Use the Feishu title format:

`YYYYMMDD <type> <specific name>`

`<type>` is a 2-4 Chinese-character descriptor such as `总结`, `探索`, or
`方案`. Treat the local Markdown file as the offline record and the Feishu
document as the online record. Include the Feishu document URL in the handoff.
